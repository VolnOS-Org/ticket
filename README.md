## What is VolnOS?

VolnOS is a general-purpose Linux operating system, built on **Ubuntu 24.04 LTS**, designed to be approachable for everyday users. Two things make it stand out:

- **Runs Windows apps by default.** Powered by Wine and preconfigured at the system level — double-click an `.exe` or `.msi` and it just runs, with no manual setup.
- **Safe over-the-air updates.** Updates are built right into Settings and use an **A/B partition** scheme with **automatic rollback**, so a failed or interrupted update can't leave your system broken.

On top of a stable Ubuntu LTS base, that means broad hardware support, familiar tooling, and updates you can trust.

Learn more on the homepage: **https://volnos.org**

## How to use this repo

We keep two channels, each for a clear purpose. Picking the right one helps us respond faster.

### Discussions — questions, ideas, feedback
Use [**Discussions**](https://github.com/VolnOS-Org/ticket/discussions) for anything open-ended:
- Questions about installing or using VolnOS
- "Does app X run under Wine on VolnOS?" reports and tips
- Feature ideas you'd like to talk through before filing
- General feedback, show-and-tell, and community help

### Issues — bugs and concrete requests
Use [**Issues**](https://github.com/VolnOS-Org/ticket/issues/new/choose) for trackable, specific items:
- A reproducible bug
- A well-defined feature or improvement request
- A documentation problem

**Before opening an issue**, please:
1. **Search existing issues** to avoid duplicates — a reaction on an existing one helps us prioritize.
2. Check the [Discussions](https://github.com/VolnOS-Org/ticket/discussions) in case it's already being talked about.

**A good bug report includes:**
- VolnOS version (Settings → About, or `cat /etc/os-release`)
- Hardware (CPU/GPU, machine model) and whether it's bare metal or a VM
- Clear steps to reproduce, and what you expected vs. what happened
- For Windows-app problems: the app name + version, and the [WineHQ AppDB](https://appdb.winehq.org/) rating if you know it
- Relevant logs or screenshots

## How you can help

Even though VolnOS development happens within the core team, **users shape the project**. The most valuable things you can do:

- **Report bugs and test fixes** — even confirming "I see this too" is valuable.
- **Request features** and tell us what would make VolnOS better for you.
- **Share Wine compatibility results** — tell us which Windows apps work (and which don't) so we can build a trusted "supported apps" list.
- **Help other users** in Discussions.
- **Report documentation gaps** — unclear steps, missing guides, typos on the site.
- **Spread the word** — write about VolnOS, or tell a friend who'd find it useful.

No question is too small — don't hesitate to ask.
